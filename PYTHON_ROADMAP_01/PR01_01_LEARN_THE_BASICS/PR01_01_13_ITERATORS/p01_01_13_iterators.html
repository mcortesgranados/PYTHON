Iterators in Python: A Comprehensive Example

Iterators are a fundamental concept in Python that allow you to efficiently process elements in a sequence one at a time without storing the entire sequence in memory. This is particularly useful for large datasets where loading everything at once might be impractical.

1. Creating a Simple Iterator

Python Code:

class NumberIterator:
    """
    An iterator class that generates a sequence of numbers.

    This class implements the iterator protocol by defining the `__iter__` 
    and `__next__` methods.
    """

    def __init__(self, start, stop):
        """
        Initializes the iterator with a starting and stopping value.

        Args:
            start (int): The starting number of the sequence.
            stop (int): The ending number (not inclusive) of the sequence.
        """
        self.current = start - 1  # Initialize before start for first __next__ call
        self.stop = stop

    def __iter__(self):
        """
        Returns the iterator object itself.

        This method is required by the iterator protocol to allow usage in for loops.
        """
        return self

    def __next__(self):
        """
        Returns the next element in the sequence.

        This method is called repeatedly in a for loop until it raises a 
        StopIteration exception, signaling the end of the sequence.
        """
        self.current += 1
        if self.current < self.stop:
            return self.current
        else:
            raise StopIteration

# Create an iterator object
number_iterator = NumberIterator(1, 6)

# Use the iterator in a for loop
for num in number_iterator:
    print(num)

Explanation:

  • We define a class NumberIterator that implements the iterator protocol.
  • The __init__ method initializes the starting and stopping values for the sequence.
  • The __iter__ method simply returns the self object, making the class iterable.
  • The __next__ method is responsible for returning the next element in the sequence.
  • It increments the current value and checks if it's less than stop.
  • If it is, it returns the current value. Otherwise, it raises StopIteration to signal the end.
  • We create an instance of NumberIterator and iterate through it using a for loop.
  • The loop calls the __next__ method repeatedly until StopIteration is raised.
